#dotfiles

these are my configuration files for linux

## setup

```bash
cp ~/.config/i3/config ~/.config/i3/config.bak
cp ./i3/config ~/.config/i3/config
cp -r ./i3status ~/.config/
cp ./90-touchpad.conf /etc/X11/xorg.conf.d/
cp -r ./fonts/ ~/.local/share/
```
